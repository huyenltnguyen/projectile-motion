# Projectile Motion

An interactive projectile motion simulator built with React, TypeScript, and Vite.

## Setup

**Prerequisites:** [Node.js](https://nodejs.org/) and [pnpm](https://pnpm.io/).

```bash
pnpm install
```

## Development

```bash
# Start the dev server
pnpm dev

# Run tests
pnpm test

# Run tests once (CI mode)
pnpm test:run

# Type-check and build
pnpm build

# Lint
pnpm lint
```

## Deploy to GitHub Pages

This repo is configured to deploy automatically from the `main` branch using GitHub Actions.
